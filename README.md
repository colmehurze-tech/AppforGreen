# AppforGreen
An electricity tracking app built using python. 

This app tracks the electricity usage on a specified power outlet using python and arduino.

Download and run now:
[Linux](https://raw.githubusercontent.com/colmehurze-tech/AppforGreen/main/dist/main) 

If you want to run it from the source code, make sure you have python and tkinter installed. Simply clone the git repo and run "python3 main.py"
